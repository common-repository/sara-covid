== Description ==

The main propose of this plugin is to view COVID-19 case statistics in worldwide. you can easily install the plugin and use the shortcode to view the data on their website.our latest version with more features will be coming soon. This plugin is free to use.

DR SARA COVID plugin communicates with an API. This API does not send any personal or identifiable information.source data from Johns Hopkins University, the New York Times, and Worldometers to give you a comphrehensive view of the data.
Privacy Policy:https://github.com/NovelCOVID/API/blob/master/privacy.md

API Information:
* Data can be updated after updated in https://disease.sh/v2/
* Link of API https://disease.sh/v2/
